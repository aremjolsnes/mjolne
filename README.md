# Mjølne - Ares tumleplass

Personal landing page for Are Mjølsnes, collecting digital experiments and projects.

## How to Run Locally

You can run this project locally using Python (pre-installed on most systems) or Node.js.

### Using Python
Run the following command in your terminal:
```bash
python -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your browser.

### Using Node.js (npx)
If you have Node.js installed, you can use:
```bash
npx serve
```

## Notes
- Some links (like `/oppskrifter/`) point to absolute paths that may not work correctly on localhost if the target folders are not present in this repository.
